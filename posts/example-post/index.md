# Welcome to My Blog

This is my first blog post using the lightweight markdown blog system.

## Features

This blog system supports:

- **Markdown formatting** with GitHub Flavored Markdown
- **Syntax highlighting** for code blocks
- **Cross-linking** between articles using [[wikilinks]]
- **Tags** for organizing content
- **Automatic publishing** on Git push

## Code Example

Here's a simple TypeScript example:

```typescript
function greet(name: string): string {
  return `Hello, ${name}!`;
}

console.log(greet('World'));
```
